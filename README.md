# Marvel App

Busca a tus personajes favoritos del Universo Marvel

**Agregar .env**

```
REACT_APP_MARVEL_PUBLIC_KEY=********************
REACT_APP_MARVEL_PRIVATE_KEY=****************
```

**Instalar depedendencias**

```
npm install
```

**Ejecutar en Local**

```
npm start
```
